# 小型编译器课程设计

##样板语言定义
```
{
       int a; int b;
       {  a=10;
          b=5;
          if (a>b) then
            b=a
          else
            a = b;
          while(a < b) do{
                a = a-b;
                b = a+b
          }
       }
}#
```

##工作流程
1. 词法分析->生成Word单词
2. 语法分析->生成语法树
3. 语法制导翻译->生成符号表和常量表
4. 语义分析和代码生成->生成四元式

##要点
- 其中，语义分析阶段主要采用了递归下降子程序法，同时进行了语法制导翻译
